# Data

## Data Source

Downloaded from Kaggle: [Spotify Dataset 1921-2020 160k Tracks](https://www.kaggle.com/yamaerenay/spotify-dataset-1921-2020-160k-tracks)

## Setup

1. Download the ZIP file from the Kaggle link above
2. Place the downloaded file in `data/raw/`
3. Unzip locally

## Note

Dataset files are not included in this repository to keep the project lightweight and respect licensing terms.
